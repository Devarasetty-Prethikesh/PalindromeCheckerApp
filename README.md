# PalindromeCheckerApp

## Objective
To create a console-based Java application to check whether a string is palindrome under different conditions.


## Use Case 2
Checks whether a hardcoded string is a palindrome by reversing the string and comparing it with the original value.

## Logic Used in UC2
- Store a predefined string in a variable
- Reverse the string using a `for` loop
- Compare original and reversed string using `equals()`
- Display result using `if-else` condition
